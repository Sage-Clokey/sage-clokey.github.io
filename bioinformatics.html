orf-finder Sequence Analysis

A command-line tool that searches all 6 reading frames of a DNA sequence for open reading frames — 3 forward and 3 reverse complement. Generates the reverse complement strand in-place, then scans for configurable start and stop codons. Handles dangling ends (ORFs that reach sequence boundaries without a stop codon), enforces a minimum ORF length, and outputs results sorted by frame and position. Built around a reusable FastAreader + NucParams library.

Python argparse FASTA 6-frame translation reverse complement OOP
GitHub
trna-essential-fragment-finder Sequence Analysis

Identifies the minimal essential unique fragments of tRNA sequences — the shortest substring that uniquely identifies each tRNA molecule among an entire mitochondrial tRNA set. Uses a power-set approach: generates all substrings of a target tRNA, builds a union of all substrings from every other tRNA, then finds the shortest substrings present only in the target. Applied to the full Bos taurus mitochondrial tRNA collection (22 sequences). Outputs one file per tRNA with its essential fragments.

Python power-set algorithm FASTA tRNA mitochondrial genomics probe design

scientific-data-visualization Featured Data Visualization

Seven weeks of publication-quality scientific figures built from scratch in Python and matplotlib — the kind used in real genomics papers. Highlights include: sequence logos computed via Shannon entropy information content (R = log₂(4) − H) with per-position nucleotide image stacking; a full genome browser that parses GTF gene annotation and PSL RNA-seq alignment files, then stacks reads by position to avoid overlap; and expression phase/heatmap panels. Every figure is built using low-level matplotlib patches — no seaborn shortcuts.

Python matplotlib numpy Shannon entropy GTF parsing PSL parsing sequence logos genome browser

genome-nucleotide-analyzer Genomics

Reads genome-scale FASTA files and reports sequence length, GC content, and per-codon usage frequencies across the entire genome. A second tool compares nucleotide composition between two genomes side-by-side. Built on a shared NucParams class that handles codon counting, DNA-to-RNA conversion, and amino acid grouping. The architecture separates I/O (FastAreader), computation (NucParams), and output formatting cleanly.

Python FASTA codon usage GC content genome analysis OOP

protein-isoelectric-point-calculator Protein Biochemistry

Calculates the isoelectric point (pI) and net charge of a protein from its amino acid sequence. Uses experimentally determined pKa values for all ionizable residues (Asp, Glu, Cys, Tyr, His, Lys, Arg) plus N- and C-terminal contributions. Implements binary search over pH space to find the charge-neutral point. Complemented by a ProteinParam class that also computes molecular weight and extinction coefficient.

Python pKa values isoelectric point protein biochemistry binary search OOP
