README

Population Genetics Monte Carlo Simulator

A Wright-Fisher population genetics simulator using Monte Carlo methods to model allele frequency dynamics across thousands of generations. Explores the interplay between genetic drift, selection pressure, and initial allele frequency across different population sizes.

What This Does

Simulates allele frequency trajectories across 10,000 generations under the Wright-Fisher model — the standard framework for modeling how gene variants spread or disappear in finite populations.

Two population sizes are compared: - N = 10,000 individuals — large population where selection dominates - N = 500 individuals — small population where genetic drift dominates

Three initial allele frequencies are tested at each population size: - p₀ = 0.1 — rare allele, starting from low frequency - p₀ = 0.5 — neutral start, equal frequency - p₀ = 0.9 — common allele, starting from high frequency

Key Biological Insight

At small population sizes (N=500), random genetic drift overwhelms selection pressure. Alleles fix or go extinct not because of fitness — but because of chance. At large population sizes (N=10,000), trajectories are smoother and selection has room to act.

This demonstrates one of the most counterintuitive results in evolutionary biology: evolution is not purely optimization. It is optimization plus noise, weighted by population size.

Repository Contents

File Pattern Description
*AF_rawdata.csv Raw allele frequency trajectory data per simulation run
*.pdf Visualization plots of allele frequency over 10,000 generations
*.html Interactive visualization (500 individuals, p₀ = 0.5)

File naming convention: {N}ind_{G}gen_{p0}AF - 10Kind = N of 10,000 individuals - 500ind = N of 500 individuals - 10Kgen = 10,000 generations - 0p1 / 0p5 / 0p9 = initial allele frequency (0.1, 0.5, 0.9)

Background

The Wright-Fisher model assumes: - Discrete, non-overlapping generations - Fixed population size N - Random sampling of 2N alleles each generation - No mutation, migration, or recombination (unless specified)

Genetic drift is the random fluctuation in allele frequency due to finite population size. Its strength is inversely proportional to N — the smaller the population, the more powerful the drift.

function processFile(file) {
  fileInfo.textContent = `📄 ${file.name}`;
  Papa.parse(file, {
    header: true, skipEmptyLines: true,
    complete: res => {
      parsedRows = parseRows(res.data);
      if (parsedRows.length) {
        fileInfo.textContent += `  —  ${parsedRows.length} simulation(s) ready`;
        runBtn.disabled = false;
      } else {
        fileInfo.textContent += '  ⚠ No valid rows found. Check column names.';
      }
    },
    error: err => { fileInfo.textContent = '⚠ Parse error: ' + err.message; }
  });
}

function parseRows(data) {
  return data.reduce((acc, row) => {
    const k = Object.keys(row).map(x => x.trim().toLowerCase());
    const v = Object.values(row).map(x => x.trim());
    const get = (...terms) => {
      for (const t of terms) {
        const i = k.findIndex(x => x.includes(t));
        if (i !== -1) return v[i];
      }
      return null;
    };
    const name   = get('run','name','sim','id') || `Sim_${acc.length+1}`;
    const nRaw   = get('population','pop size','pop_size');
    const genRaw = get('generation','gen');
    const p0Raw  = get('starting','frequency','freq','p0');
    if (!nRaw || !genRaw || !p0Raw) return acc;
    const N   = parseN(nRaw);
    const gen = parseInt(genRaw, 10);
    const p0  = parseFloat(p0Raw);
    if (N === null || isNaN(gen) || isNaN(p0) || p0 < 0 || p0 > 1) return acc;
    acc.push({ name, N, generations: gen, p0 });
    return acc;
  }, []);
}

function parseN(raw) {
  const s = String(raw).toLowerCase().trim();
  if (['infinity','infinite','inf','∞'].includes(s)) return Infinity;
  const n = parseInt(s, 10);
  return isNaN(n) ? null : n;
}

// ================================================================
// SIMULATION — Wright-Fisher model
// ================================================================
function normalRnd() {
  let u = 0, v = 0;
  while (!u) u = Math.random();
  while (!v) v = Math.random();
  return Math.sqrt(-2 * Math.log(u)) * Math.cos(2 * Math.PI * v);
}

function binomial(n, p) {
  if (p <= 0) return 0;
  if (p >= 1) return n;
  if (n > 200) {
    const s = Math.max(0, Math.min(n, Math.round(n*p + Math.sqrt(n*p*(1-p)) * normalRnd())));
    return s;
  }
  let c = 0;
  for (let i = 0; i < n; i++) if (Math.random() < p) c++;
  return c;
}

function simulate({ name, N, generations, p0 }) {
  const inf = !isFinite(N);
  const af  = new Float64Array(generations + 1);
  af[0] = p0;
  let p = p0, fixGen = -1, lossGen = -1;

  for (let g = 1; g <= generations; g++) {
    if (inf || p <= 0 || p >= 1) {
      af[g] = p;
    } else {
      p = binomial(2 * N, p) / (2 * N);
      af[g] = p;
    }
    if (fixGen  < 0 && p >= 1) fixGen  = g;
    if (lossGen < 0 && p <= 0) lossGen = g;
  }

  // Genotype frequencies (HWE each gen)
  const g11 = af.map(q => q * q);
  const g12 = af.map(q => 2 * q * (1 - q));
  const g22 = af.map(q => (1-q) * (1-q));

  const finalFreq = af[generations];
  const avg = af.reduce((a,b) => a+b, 0) / af.length;
  const sd  = Math.sqrt(af.reduce((a,b) => a + (b-avg)**2, 0) / af.length);

  const outcome = inf ? 'stable'
    : finalFreq >= 1 ? 'fixed'
    : finalFreq <= 0 ? 'lost'
    : 'polymorphic';

  return { name, N, generations, p0, inf, af, g11, g12, g22, fixGen, lossGen, finalFreq, avg, sd, outcome };
}

function sample(arr, max) {
  if (arr.length <= max) return { v: Array.from(arr), l: arr.map((_,i)=>i) };
  const step = Math.ceil(arr.length / max);
  const v = [], l = [];
  for (let i = 0; i < arr.length; i += step) { v.push(arr[i]); l.push(i); }
  if (l[l.length-1] !== arr.length-1) { v.push(arr[arr.length-1]); l.push(arr.length-1); }
  return { v, l };
}

function generateAnalysis(results) {
  const byP0   = {};
  for (const r of results) {
    const k = r.p0.toFixed(3);
    (byP0[k] = byP0[k] || []).push(r);
  }

  const finite   = results.filter(r => isFinite(r.N));
  const infinite = results.filter(r => !isFinite(r.N));
  const allNs    = [...new Set(finite.map(r => r.N))].sort((a,b) => a-b);
  const allGens  = [...new Set(results.map(r => r.generations))];
  const allP0s   = Object.keys(byP0);
  let html = '';

  // Overview
  html += `<div class="analysis-section">
    <h3>Overview</h3>
    <p>
      <strong>${results.length} simulation${results.length>1?'s':''}</strong> were run under neutral conditions
      (no selection, mutation, migration, inbreeding, or assortative mating).
      Starting A₁ frequencies tested: ${allP0s.map(k=>`p₀ = ${k}`).join(', ')}.
      Population sizes: ${[...new Set(results.map(r=>fmtN(r.N)))].join(', ')}.
      Duration: ${allGens.map(g=>g.toLocaleString()).join(' or ')} generations.
    </p>
  </div>`;

  // Q1 — drift vs N
  if (finite.length) {
    html += `<div class="analysis-section">
      <h3>Effect of Population Size on Genetic Drift</h3>
      <p>
        Genetic drift is the random fluctuation of allele frequencies caused by chance variation in
        which individuals reproduce. Its strength depends directly on population size through the
        per-generation sampling variance:
      </p>
      <div class="formula">Var(Δp) = p(1 − p) / 2N</div>
      <p>
        As N decreases, the variance increases, producing larger random swings in allele frequency
        each generation and accelerating fixation or loss.
      </p>`;

    for (const [p0k, grp] of Object.entries(byP0)) {
      const fg = grp.filter(r => isFinite(r.N)).sort((a,b) => a.N - b.N);
      if (fg.length < 1) continue;
      html += `<div class="hl"><strong>At p₀ = ${p0k}:</strong><ul>`;
      for (const r of fg) {
        let desc;
        if      (r.outcome==='fixed') desc = `A₁ <strong>fixed</strong> at generation ${r.fixGen.toLocaleString()} (SD = ${r.sd.toFixed(4)}) — rapid drift to fixation`;
        else if (r.outcome==='lost')  desc = `A₁ <strong>lost</strong> at generation ${r.lossGen.toLocaleString()} (SD = ${r.sd.toFixed(4)}) — rapid drift to loss`;
        else                          desc = `Remained <strong>polymorphic</strong> through generation ${r.generations.toLocaleString()} (final freq = ${r.finalFreq.toFixed(3)}, SD = ${r.sd.toFixed(4)}) — drift too weak to cause fixation`;
        html += `<li><strong>N = ${r.N.toLocaleString()}:</strong> ${desc}</li>`;
      }
      html += `</ul></div>`;
    }

    if (allNs.length > 1) {
      html += `<p>The frequency standard deviation (SD) directly quantifies drift intensity. `;
      for (const n of allNs) {
        const g2 = finite.filter(r => r.N === n);
        const avgSD = g2.reduce((a,r) => a + r.sd, 0) / g2.length;
        html += `At N = ${n.toLocaleString()}, average SD across simulations = <strong>${avgSD.toFixed(4)}</strong>. `;
      }
      html += `Smaller populations consistently show higher SD, confirming stronger drift.</p>`;
    }
    html += `</div>`;
  }

  // Q2 — infinite population
  if (infinite.length) {
    html += `<div class="analysis-section">
      <h3>Infinite Population: Hardy-Weinberg Equilibrium</h3>
      <p>
        When N = ∞, the sampling variance per generation approaches zero:
      </p>
      <div class="formula">Var(Δp) = p(1 − p) / 2N → 0 as N → ∞</div>
      <p>
        With no sampling error, allele frequencies remain exactly constant — the defining condition
        of Hardy-Weinberg equilibrium. Genotype frequencies stay fixed at p² (A₁A₁), 2pq (A₁A₂),
        and q² (A₂A₂) throughout all generations.
      </p>
      <div class="hl"><strong>Infinite population results:</strong><ul>`;
    for (const r of infinite) {
      html += `<li><strong>${esc(r.name)}:</strong> A₁ held exactly at p = ${r.p0} for all
        ${r.generations.toLocaleString()} generations (SD = ${r.sd.toFixed(6)}).
        Genotypes: A₁A₁ = ${(r.p0**2).toFixed(4)},
        A₁A₂ = ${(2*r.p0*(1-r.p0)).toFixed(4)},
        A₂A₂ = ${((1-r.p0)**2).toFixed(4)}.</li>`;
    }
    html += `</ul></div></div>`;
  }

  // Q3 — starting frequency
  html += `<div class="analysis-section">
    <h3>Role of Starting Allele Frequency</h3>
    <p>
      When drift is minimal and no evolutionary forces act, the <strong>starting allele frequency
      (p₀)</strong> is the primary determinant of the average final frequency. Drift is an
      unbiased process: the expected final frequency across many replicate populations always
      equals p₀, and the probability of fixation under pure drift = p₀.
    </p>`;

  if (allP0s.length > 1) {
    html += `<div class="hl"><strong>Average final frequency by starting frequency:</strong><ul>`;
    for (const [p0k, grp] of Object.entries(byP0)) {
      const avgFinal = grp.reduce((a,r)=>a+r.finalFreq,0) / grp.length;
      const fixed = grp.filter(r=>r.outcome==='fixed').length;
      const lost  = grp.filter(r=>r.outcome==='lost').length;
      const other = grp.length - fixed - lost;
      html += `<li><strong>p₀ = ${p0k}:</strong> avg final freq = ${avgFinal.toFixed(4)} &nbsp;|&nbsp; ${fixed} fixed, ${lost} lost, ${other} polymorphic/stable</li>`;
    }
    html += `</ul></div>
    <p>Higher starting frequencies yield higher average final frequencies, consistent with the
    neutral expectation that p₀ predicts long-run outcomes.</p>`;
  }
  html += `</div>`;

  // Conclusions
  html += `<div class="analysis-section">
    <h3>Conclusions</h3>
    <div class="hl"><ol>
      <li>
        <strong>Population size governs drift strength.</strong>
        Smaller populations experience larger per-generation sampling variance [Var = p(1−p)/2N],
        leading to rapid and dramatic allele frequency changes. Larger populations buffer against
        drift, keeping allele frequencies near their starting values for far longer.
      </li>
      <li>
        <strong>Infinite populations maintain Hardy-Weinberg equilibrium.</strong>
        Genetic drift requires finite sampling to operate. At N = ∞, allele and genotype
        frequencies remain perfectly stable indefinitely, satisfying HWE exactly.
      </li>
      <li>
        <strong>Starting allele frequency determines average long-run outcomes.</strong>
        In the absence of selection and with drift minimized, p₀ sets the expected final frequency
        and equals the probability of fixation, because drift is directionally neutral on average.
      </li>
    </ol></div>
  </div>`;

  return html;
}

  <h2>Discussion Questions &amp; Answers</h2>

  <!-- Q1 -->
  <h3>Q1: Why is the impact of genetic drift strongest in small, compared to large populations?</h3>

  <p>
    Genetic drift is the random change in allele frequency due to chance variation in which
    individuals reproduce in each generation. In small populations, each offspring represents
    a larger fraction of the total gene pool, so random sampling errors have a proportionally
    greater effect on allele frequencies.
  </p>

  <div class="formula">Var(Δp) = p(1 − p) / 2N</div>

  <p>
    This formula shows that the variance in allele frequency change per generation is inversely
    proportional to population size (N). As N decreases, variance increases — meaning larger
    random fluctuations occur each generation.
  </p>

  <div class="callout">
    <strong>Evidence from simulations:</strong>
    <ul>
      <li><strong>N=500, p=0.5:</strong> A1 was completely lost by generation ~2,000 — strong drift eliminated the allele in less than 20% of the total time span.</li>
      <li><strong>N=10,000, p=0.5:</strong> A1 fluctuated between 0.3–0.7 across the full 10,000 generations without ever reaching fixation or loss — drift was too weak to cause fixation.</li>
      <li><strong>N=500, p=0.1:</strong> A1 randomly surged from 0.1 to ~0.9 within ~1,000 generations, then crashed to 0 — a dramatic demonstration of large random swings possible in small populations.</li>
    </ul>
  </div>

  <div class="answer-box">
    <strong>Answer:</strong> Genetic drift is strongest in small populations because random sampling
    errors represent a larger proportion of the total gene pool each generation. With fewer
    individuals, a chance event (e.g., one allele not being passed on) has an outsized effect
    on overall allele frequency. In large populations, these random errors cancel out across
    thousands of individuals, producing minimal net change per generation.
  </div>

  <!-- Q2 -->
  <h3>Q2: Why do allele frequencies become uniform when population size is set to infinite?</h3>

  <p>
    Genetic drift is fundamentally a sampling phenomenon — it arises because a finite number
    of individuals reproduce each generation and alleles are randomly sampled. When population
    size is infinite, sampling error mathematically approaches zero.
  </p>

  <div class="formula">Var(Δp) = p(1 − p) / 2N → 0 as N → ∞</div>

  <p>
    With no variance in allele frequency change, allele frequencies cannot change from one
    generation to the next. This is the idealized condition described by the
    Hardy-Weinberg principle: in an infinitely large, randomly mating population with no
    evolutionary forces, allele and genotype frequencies remain constant indefinitely.
  </p>

  <div class="callout">
    <strong>Evidence from simulations:</strong>
    <ul>
      <li><strong>Infinind, p=0.5:</strong> Perfectly flat line at 0.5 for all 10,000 generations. Genotypes: A1A1=0.25, A1A2=0.50, A2A2=0.25 (exactly p² = 0.25, 2pq = 0.50, q² = 0.25).</li>
      <li><strong>Infinind, p=0.1:</strong> Perfectly flat line at 0.1. Genotypes: A1A1=0.01, A1A2=0.18, A2A2=0.81.</li>
      <li><strong>Infinind, p=0.9:</strong> Perfectly flat line at 0.9. Genotypes: A1A1=0.81, A1A2=0.18, A2A2=0.01.</li>
      <li>In all three infinite cases, both allele and genotype frequency graphs were completely flat horizontal lines — no variation whatsoever across 10,000 generations.</li>
    </ul>
  </div>

  <div class="answer-box">
    <strong>Answer:</strong> With an infinite population size, every possible allele is transmitted
    in exact proportion each generation because there is no sampling error. Genetic drift, which
    requires a finite sample, cannot occur. Allele frequencies therefore remain perfectly stable
    ("uniform") across all generations, exactly matching their starting values and satisfying
    Hardy-Weinberg equilibrium.
  </div>

  <!-- Q3 -->
  <h3>Q3: When no evolution is occurring and genetic drift is minimal, what factor has the largest impact on the average final allele frequency of A1?</h3>

  <p>
    When drift is minimized (large or infinite N) and no evolutionary forces act (neutrality),
    the expected final allele frequency always equals the starting frequency. This is because
    drift is an <em>unbiased</em> process — it has no directional preference. On average across
    many replicate populations, the mean final frequency equals p₀ regardless of the number
    of generations.
  </p>

  <div class="callout">
    <strong>Evidence from simulations (N=∞ and N=10,000):</strong>
    <table style="margin-top:8px;">
      <thead>
        <tr>
          <th>Simulation</th>

          <th>Approx. Final Freq</th>
        </tr>
      </thead>
      <tbody>
        <tr><td>Infinind_0p1AF</td><td>0.1</td><td>0.1 (unchanged)</td></tr>
        <tr><td>Infinind_0p5AF</td><td>0.5</td><td>0.5 (unchanged)</td></tr>
        <tr><td>Infinind_0p9AF</td><td>0.9</td><td>0.9 (unchanged)</td></tr>
        <tr><td>10Kind_0p1AF</td><td>0.1</td><td>~0 (lost — started low, drift caused loss)</td></tr>
        <tr><td>10Kind_0p5AF</td><td>0.5</td><td>~0.5–0.6 (near starting freq)</td></tr>
        <tr><td>10Kind_0p9AF</td><td>0.9</td><td>~0.95–1.0 (near starting freq)</td></tr>
      </tbody>
    </table>
    <p style="margin-top:8px;">
      Note: The N=10,000 p=0.1 result (loss) is consistent with the starting frequency — a
      rare allele (p=0.1) has only a 10% probability of fixation and a 90% probability of
      loss under neutrality, matching the expected outcome.
    </p>
  </div>

  <div class="answer-box">
    <strong>Answer:</strong> The <strong>starting allele frequency (p₀)</strong> has the largest
    impact on the average final allele frequency of A1. When drift is minimal and no evolutionary
    forces act, expected final frequency = p₀. The starting frequency also determines the
    probability of fixation under drift (probability of fixation = p₀), explaining why alleles
    starting at high frequency (p=0.9) are far more likely to become fixed than alleles starting
    at low frequency (p=0.1).
  </div>
