Scientific Data Visualization

Publication-quality scientific figures built with Python and matplotlib, from UC Santa Cruz BME 163 (Scientific Visualization).

Each assignment builds progressively more complex visualization techniques used in real bioinformatics research.

Assignments

Week File Visualization Type
1 week1/ Geometric shapes, color interpolation, heatmaps
2 week2/ Scatter plots and line graphs from tabular data
3 week3/ Cell type expression data with color gradients
4 week4/ Enhanced multi-panel figures
5 week5/ Sequence logos — splice site information content using Shannon entropy
6 week6/ Gene expression phase plots
7 week7/ Multi-panel composite figures
Final final/ Full genome browser: GTF gene models + PSL RNA-seq read alignments stacked by position

Highlights

Week 5 — Sequence Logo: - Parses FASTA splice sequences - Computes per-position base frequencies - Calculates information content using Shannon entropy: R = log2(4) − H - Renders nucleotide images scaled by information content at each position

Final — Genome Browser: - Parses GTF annotation files for gene models (CDS, exon blocks) - Parses PSL alignment files for RNA-seq reads - Implements a stacking algorithm to display reads without overlap - Renders multi-track figure: gene models on top, paired-end reads below

    def checkOverlap(self,position, pointListDict,point_size = 2):
        xrange = self.xrange
        yrange = self.yrange
        panelWidth = self.panelWidth
        panelHeight = self.panelHeight
        x_position, y_position = position  # Unpack the position tuple
        for key, value in pointListDict.items():
            other_x_position, other_y_position = value
            
                
            # Calculate the distance between the two positions
            xdist = ((x_position - other_x_position)/xrange)*panelWidth
            ydist = ((y_position - other_y_position)/yrange)*panelHeight
            distance = (xdist ** 2 + ydist** 2) ** 0.5

            # Check if the distance is less than or equal to the point size
            if distance < point_size/72:
                return True
        # Returns that there is no overlap
        return False

        """

        Parameters:

        - color2: tuple representing the RGB values of the second color (r, g, b)
        - steps: number of steps for interpolation

        Returns:

        """
        step_r = (color2[0] - color1[0]) / steps
